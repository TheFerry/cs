# 依赖
> - 字体: 任意一种nerd font
> - 编译器: 支持c++17的编译器

# 安装
> 1. 克隆本项目
`git clone https://github.com/semtor/cs.git`
> 2. 进入到项目录
`cd cs`
> 3. 编译源码
`sudo make install`
